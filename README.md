# Realtime Chat by Ayush

## Description
A real-time chat application built with Node.js, Express, Socket.io, HTML, CSS, and Vanilla JS.

## Features
- Users can enter their name and join the chat.
- Messages are broadcast to all connected users in real time.
- System messages for user join/leave events.
- Responsive UI for mobile and desktop.
- Modern CSS design without frameworks.
- Messages are aligned based on the sender.

## Installation
1. Clone the repository.
2. Navigate to the project directory.
3. Run `npm install` to install dependencies.
4. Start the server with `npm start`.

## Deployment
Follow the AWS EC2 deployment guide to set up the application on an EC2 instance.
